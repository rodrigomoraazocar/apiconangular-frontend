FrontendAngular

Este projeto foi gerado com o Angular CLI versão 13.2.6.

Servidor de desenvolvimento

Execute ng serve para um servidor de desenvolvimento. Navegue até http://localhost:4200/. O aplicativo recarregará automaticamente se você alterar algum dos arquivos fonte.

Mais ajuda

Para obter mais ajuda sobre o Angular CLI, use ng help ou confira a página de Visão geral e referência do Angular CLI.
